# Niugini Tech Storefront (Static)

This is a **static** storefront (HTML/CSS/JS) that works on **GitHub Pages** with **no terminal** required.

## Features
- Product catalog loaded from `products.json`
- Cart stored in browser `localStorage`
- Checkout generates a WhatsApp or Email order message

## Edit products
Open `products.json` and change:
- name
- price_kina
- category
- desc
